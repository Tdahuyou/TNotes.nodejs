## 1. 📒 概述

| 方法名    | 功能描述                                   |
| --------- | ------------------------------------------ |
| `stat()`  | 异步查看目录或文件信息（包括文件大小等）   |
| `lstat()` | 异步查看符号链接本身的信息（而非目标文件） |
